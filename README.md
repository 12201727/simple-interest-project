# Simple Interest Calculator

This project provides a simple Bash script to calculate **Simple Interest**.

## Formula
Simple Interest (SI) = (P × R × T) / 100  
Where:  
- **P** = Principal Amount  
- **R** = Rate of Interest  
- **T** = Time (in years)  

## How to Run
```bash
chmod +x simple-interest.sh
./simple-interest.sh
